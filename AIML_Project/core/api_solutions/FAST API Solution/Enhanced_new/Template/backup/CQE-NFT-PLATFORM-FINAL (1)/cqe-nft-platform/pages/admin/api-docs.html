📖 API Documentation

Complete API reference for CQE NFT Platform

🔐 Authentication

POST/api/v1/auth/login

Description: Authenticate user and get session token

Request Body:

{
  "username": "[email protected]",
  "password": "password123",
  "totp": "123456"
}

Response:

{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "email": "[email protected]",
    "role": "admin"
  }
}
POST/api/v1/auth/logout

Description: Invalidate current session

📊 Test Result Analysis

GET/api/v1/analysis/releases?lob={lob}

Description: Get all releases for a specific LOB

GET/api/v1/analysis/tests?lob={lob}&release={release}

Description: Get all tests for a LOB and release

GET/api/v1/analysis/report/{pc_run_id}

Description: Get comprehensive test report with all data sources

⚡ Monitoring

GET/api/v1/monitoring/tracks?lob={lob}

Description: Get all tracks for a LOB

GET/api/v1/monitoring/track/config?track={track}

Description: Get track template configuration

POST/api/v1/monitoring/start

Description: Start monitoring session

Request Body:

{
  "pc_run_id": "65989",
  "test_name": "Peak Load Test",
  "release_name": "Release 2.5",
  "lob": "Digital Technology",
  "track": "CDV3",
  "duration": 1800
}
POST/api/v1/monitoring/stop

Description: Stop monitoring session and get data summary

👤 Admin Operations

DELETE/api/v1/admin/test-data/{pc_run_id}

Description: Delete all data for a PC Run ID from all sources

GET/api/v1/admin/tracks

Description: Get all track templates

PUT/api/v1/admin/tracks/{track_id}

Description: Update track template configuration

POST/api/v1/admin/users/create

Description: Create new user

GET/api/v1/admin/audit-logs

Description: Get system audit logs with filters
